# SPEED

Explored low-level C++ performance for fun by building faster versions of common algorithms and data structures, including GCD, binary search, and hash tables. Beat STL baselines by up to 2x on GCD, 8x on binary search, and 5x on hash table lookups using branch elimination, SIMD, CPU intrinsics, prefetching, and cache-aware layouts.

## Layout

- `algorithms/`: GCD, binary search, and hash table experiments.
- `performance/`: Branching, CPU cache, SIMD, and profiling experiments.
